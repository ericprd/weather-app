## Development

Clone project

```sh
git clone github.com/ericprd/weather-app
```

Install dependencies:

```sh
npm install
```

Run server in development mode:

```sh
npm run dev
```

Build for production:

```sh
npm run build
```

Start in production mode:

```sh
npm run preview
```
